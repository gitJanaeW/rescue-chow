# rescue-chow

## Purpose

With Rescue Chow, you can treat your pet while saving pets in need! By purchasing treats from this e-commerce store, 25% of your proceeds go to an animal charity of your choice.

## Built With

- MongoDB
- Express
- React
- Node
- Stripe
- GraphQL
- Mongoose
- Apollo
- JWTs
- Bcrypt
- Concurrently
